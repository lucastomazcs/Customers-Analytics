# Customers-Analytics
Este projeto tem como objetivo analisar o comportamento de clientes de uma empresa de telecomunicações, identificando padrões relacionados à retenção, cancelamento de serviços e geração de receita.

# Objetivos

- Analisar o perfil dos clientes da empresa
- Identificar fatores relacionados ao churn
- Avaliar padrões de receita e cobrança
- Criar segmentações úteis para tomada de decisão
- Desenvolver dashboards executivos para apresentação dos resultados

# 1. Perguntas de Negócio:
- Qual é a taxa geral de churn da empresa?
- Quais tipos de contrato apresentam maior taxa de churn?
- Clientes com maior tempo de relacionamento tendem a cancelar menos?
- Qual método de pagamento está mais associado ao churn?
- Clientes com cobranças mensais mais altas cancelam mais?
- Quais perfis de clientes representam maior risco de abandono?
- Quais ações podem ser sugeridas para reduzir o churn?

# Observação durante Análise: 

- A taxa de churn varia fortemente de acordo com o tipo de contrato. Clientes com contratos mensais apresentam a maior taxa de cancelamento, com aproximadamente 42,7%, enquanto clientes com contratos anuais possuem churn de 11,3% e clientes com contratos de dois anos apresentam apenas 2,8%. Isso indica que contratos de maior duração estão associados a maior retenção de clientes.

Essa observação no momento me gera uma recomendação de:
A empresa poderia criar campanhas para incentivar clientes mensais a migrarem para planos anuais ou de dois anos, oferecendo descontos, benefícios extras ou vantagens exclusivas.
